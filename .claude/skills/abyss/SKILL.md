# 天機 — タスク委任

グローバルインストール済みの OpenAI Codex CLI (`codex exec`) を非対話モードで使用し、タスクを委任する。

## プロンプトの構築

- **ユーザーが `/abyss` で呼び出した場合**: `$ARGUMENTS` をそのままプロンプトとして使用する。
- **自動起動の場合**: 現在の会話コンテキストから、明確で自己完結したプロンプトを合成する。
  天機はこの会話にアクセスできない隔離されたセッションで実行されるため、
  必要な情報をすべてプロンプトに含めること。

## 実行

以下のコマンドを実行する。`<PROMPT>` を構築したプロンプトに置き換えること。
プロンプトはシングルクォートで囲み、内部のシングルクォートは `'\''` でエスケープすること。

```bash
codex exec \
  -m gpt-5.5 \
  -c model_reasoning_effort=xhigh \
  -s read-only \
  -C "$(pwd)" \
  '<PROMPT>'
```

重要事項:

- Bashツール呼び出しのタイムアウトは **600000ms** を指定すること。
- `--json` は使用しない。プレーンテキストのstdoutのみ取得する。
- `-o` でファイルに書き出さない。stdoutを直接読み取ること。

## 実行後の処理

- **成功 (終了コード 0)**: 天機の出力をユーザーに提示し、会話に自然に統合する。
  付加価値がある場合のみ、自身のコメントを追加する。
- **失敗 (非ゼロ終了コード)**: 受け取ったエラーをそのままユーザーに報告する。
  リトライしない。ユーザーが明示的に求めない限り、自力での解決にフォールバックしない。
